# Z-MUSIC v2

## zm2_gettrktbl
Z-MUSIC v2のファンクションコール`$3a get_trk_tbl`で絶対チャンネルテーブルと演奏トラックテーブル
を取得し、アドレスと内容を表示します。


## zm2_minp1
Z-MUSIC v2のファンクションコール`$55 m_inp1`でMIDIから1バイト入力し、結果を表示します。  
コマンドライン引数で待機時間を指定できます(省略時は0=待機なし)。


## zm2_mstat
Z-MUSIC v2のファンクションコール`$09 m_stat`で演奏状態を取得し、結果を表示します。  
コマンドライン引数でチャンネル(ビットマップ形式)を指定できます
(省略時は全チャンネル検査)。


## zm2_oddopm
奇数アドレスに配置したZ-MUSIC v2のZMDデータをOPMデバイスに書き込みます。
あらかじめzmusic.xかzmsc.xを常駐しておいてください。

MPUが68000の場合はZ-MUSIC内でアドレスエラーが発生します。


## zm2_t12poc
ZMSC.X version 2.08を常駐解除する際、常駐時の親プロセスと同じアドレスに別のメモリブロックがあると
そのメモリ内容の一部を破壊してしまう不具合を検証します。

パスの通った場所にzmsc.xを置くか、コマンドライン引数でzmsc.xのパス名を指定してください。  
(zmusic.xを指定することもできますが、該当部分のコードがzmsc.xと違うため同じ不具合はありません。)

実行するとZMSC.Xを常駐してから常駐解除し、メモリ内容が破壊されたら表示します。
