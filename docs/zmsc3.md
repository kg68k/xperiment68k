# Z-MUSIC v3

### zm3_exg_memid
Z-MUSIC v3のファンクションコール`ZM_EXCHANGE_MEMID`でメモリブロックの用途IDを変更し、結果を表示します。  
コマンドライン引数で`d1.l`(モード)、`d2.l`(更新元メモリの用途ID)、`d3.l`(希望メモリの用途ID)、
`a1.l`(メモリアドレス)の値を指定します。


### zm3_free_mem2
Z-MUSIC v3のファンクションコール`ZM_FREE_MEM2`でメモリブロックを解放し、結果を表示します。  
コマンドライン引数で削除するメモリ用途IDを指定できます(8桁の16進数、省略時は全メモリ削除)。

システム関連ワークを削除するとZ-MUSIC v3が正常動作しなくなるため、メモリ用途IDに
`ID_SYSTEM`や`ID_ALL`に対応する値を指定してはいけません。


### zm3_init
Z-MUSIC v3のファンクションコール`ZM_INIT`でドライバを初期化し、結果を表示します
(戻り値はZ-MUSICのバージョンIDです)。
